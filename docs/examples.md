# HER Prompt Examples (60+)

Use these directly in Telegram. They are short, practical, and spread across HER capabilities.

## 1) Everyday Chat (6)
1. Help me plan my day in 3 priorities.
2. I feel overwhelmed. Give me a 5-minute reset plan.
3. Rewrite this to sound confident: `I think maybe we can...`
4. Summarize this in 2 lines: `...`
5. Ask me 3 questions to clarify my goal.
6. Give me 2 options: quick fix vs long-term fix.

## 2) Memory & Continuity (6)
1. Remember that I prefer concise replies.
2. Remember my birthday is October 12.
3. What do you remember about my work goals?
4. Update memory: I now prefer morning workouts.
5. Recall my last 3 commitments and remind me of them.
6. Based on our history, what pattern should I improve?

## 3) Reminders & Scheduling (6)
1. Remind me in 20min to call Ali.
2. After 2h remind me to send the report.
3. Every day at 08:30 remind me to take vitamins.
4. Every week remind me to review finances.
5. Next Monday at 9am remind me to review roadmap.
6. Every 3 days remind me to back up my laptop.

## 4) Natural Automation Rules (6)
1. Check BTC every 2min and notify me if it rises 10% from current price.
2. Track ETH every 5min and alert me if it drops 5% from baseline.
3. Monitor this endpoint every 10min and notify if `status` is not `ok`: `https://...`
4. Every hour check this page and notify me if unavailable: `https://...`
5. Every weekday at 09:00 send me a priorities check-in.
6. Watch this JSON feed daily and notify if value > 100: `https://...`

## 5) Web Search & Live Context (6)
1. Search latest AI safety news and give 3 bullets with links.
2. Find current BTC USD price and timestamp it.
3. Compare top 3 sources on this claim: `...`
4. What changed in Python recently? Include sources.
5. Find practical guides for PostgreSQL backups.
6. Search latest release notes for Docker Desktop.

## 6) MCP Tool Usage (6)
1. Show current MCP capability status.
2. Use web tool to research `topic` and summarize sources.
3. Use sequential thinking to break down this problem: `...`
4. Fetch this PDF and extract 5 key points: `URL`
5. Check available runtime tools and explain what each can do.
6. If a tool is unavailable, explain fallback behavior.

## 7) Sandbox & Diagnostics (6)
1. Run DNS lookup for `openai.com` and explain simply.
2. Check SSL certificate expiry for `github.com`.
3. Test connectivity to `api.github.com` and report latency.
4. Inspect HTTP headers for `https://example.com`.
5. Run a basic port scan summary for `host`.
6. Run traceroute-style diagnostics and summarize hops.

## 8) Productivity & Planning (6)
1. Turn this goal into a 7-day action plan: `...`
2. Break this project into milestones, risks, and owners.
3. Convert these notes into actionable tasks: `...`
4. Build a daily deep-work schedule around my meetings.
5. Create a weekly review template for me.
6. Make a shutdown checklist for end-of-day.

## 9) Communication Coaching (6)
1. Draft a polite follow-up for this message: `...`
2. Make this feedback assertive but respectful: `...`
3. Give 3 versions: formal, neutral, friendly.
4. Role-play negotiation with me in 5 turns.
5. Improve clarity of this technical explanation: `...`
6. Write a concise status update from these bullet points: `...`

## 10) Reflection & Growth (6)
1. Help me reflect on today: wins, misses, next step.
2. Ask me 5 questions for weekly self-review.
3. Turn this failure into lessons and a new plan: `...`
4. Track one habit for 14 days and check in daily.
5. Spot self-sabotage patterns in my notes: `...`
6. Build an accountability routine for my top goal.

## Command Shortcuts
- `/example` shows topic overview.
- `/example all` prints all built-in Telegram examples.
- `/example <topic>` where topic is one of:
  - `chat`, `memory`, `scheduling`, `automation`, `web`, `mcp_tools`, `sandbox`, `admin`, `personality`, `productivity`
